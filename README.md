# tick-counter web demo

A minimum implementation of the
[tick-counter library](https://github.com/hidden-primary-net/js-tick-demo-01-lib).

The [package.json](package.json) expects the library in a directory `../01-lib`
parallel to this repo directory!

## Start the server

```sh
npm install
npm run dev
```

It should look similar to this:

```
$ npm run dev

> tick-counter-web-demo@0.0.1 dev
> vite

  VITE v7.3.1  ready in 225 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help
```

The service should then be accessible: http://localhost:5173/
